Projects

Canvas - Drive Sync

December 2017 - January 2018

A Google Chrome extension that I built during my winter break to list instructor uploaded files from UC Davis' Canvas LMS. The goal of this extension was to provide an alternate interface to quickly browse files for a course. This allowed for the ability to easily open files such as PDFs and images natively in the browser as new tabs, which was not possible through the Canvas LMS user interface. The extension also integrates with Google Drive to allow files to be synced from Canvas LMS to Google Drive. I built this extension using HTML, CSS, and JavaScript. Data is obtained via REST APIs of Canvas LMS and Google Drive.

SJACraft II: MOSS of Darkness

September 2017 - December 2017

An Android port of Blizzard Entertainment's Warcraft II: Tides of Darkness. This app was a project for Professor Christopher Nitta's Software Engineering course (ECS160), and is based on his Linux port of the game written in C++. I was primarily responsible for implementing the camera, menu screens, and button layout. Utilized Java, libGDX (a Java framework for cross-platform game development), and Android Studio (built on IntelliJ IDEA). Contributors for this project are AJ Collins (Team Lead), Hung Mai, Ian Skebba, Kevin Ow, Kim Dang, Kimi Tanaka, Manohar Boppana, Sally Ly, Sasha Sirovica, and Sven Olson.

Download APK. Compatible with devices running Android 4.4 KitKat (API level 19) and newer. Please note that the map will start all black. Simply zoom out to find the peasant. Buttons on the side bar during game play need to be held down while being used. Please feel free to email me if you would like to take a look at the source code.

Photobooth

April 2017 - June 2017

My first web application! Developed as part of Professor Nina Amenta's Web Programming Course (ECS162, formerly ECS189H), it allows users to upload pictures and automatically generate tags based on the image content, which is powered by Google's Cloud Vision API. Users can also delete, edit, or add tags (up to 10); filter by tags; and mark their favorite photos. This site is implemented using HTML, CSS, JavaScript, and Node.js. Node modules used include Express and SQLite. Collaborated with Bryan Kim and Jason Hui. Assets and design courtesy of Dani Macedo.
